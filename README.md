# Praxis Malinowski – Neue Webseite

Moderne, Apple-inspirierte Webseite für die Praxis für Psychotherapie Veronika Malinowski.

## Bilder austauschen (sehr wichtig!)

Im Ordner `images/` liegen aktuell **stilisierte Platzhalter** mit ruhiger Atmosphäre.
Diese sollten durch **echte Fotos vom Kamphof** ersetzt werden.

### Bildquellen
Sehr empfehlenswert: Direkt von **kamphof-krefeld.de** (Eigentum von Jochen / Malino GmbH).
Die Bilder dort sind in sehr guter Qualität und thematisch perfekt passend.

### So gehst du vor

1. Öffne https://kamphof-krefeld.de/ im Browser
2. Rechtsklick auf das gewünschte Bild → **"Bild speichern unter…"**
3. Speichere es mit dem **exakten Dateinamen** aus der Tabelle unten in den `images/`-Ordner
4. Fertig — die Webseite zeigt das Bild automatisch an

### Empfohlene Zuordnung

| Datei in `images/`     | Was sollte da hin                                                | Vorschlag vom Kamphof                  |
| ---------------------- | ---------------------------------------------------------------- | -------------------------------------- |
| `hero.jpg`             | Großes Bild auf der Startseite (1920×1200)                       | `Rueckansicht.jpg` oder `Garten.jpg`   |
| `karte-gesetzlich.jpg` | Karte 1 auf Startseite (900×700)                                 | `IMG_9938.jpg`                         |
| `karte-privat.jpg`     | Karte 2 auf Startseite (900×700)                                 | `IMG_9945.jpg` oder `Kueche2.jpg`      |
| `karte-selbstzahler.jpg` | Karte 3 auf Startseite (900×700)                               | `IMG_9947.jpg` oder `Garten3.jpg`      |
| `feature-1.jpg`        | Feature-Block Startseite (1000×1200, Hochformat)                 | `IMG_9933.jpg` oder `Eingang.jpg`      |
| `praxis-main.jpg`      | Hero auf Praxis-Seite (1920×900, Panorama)                       | `Rueckansicht.jpg`                     |
| `praxis-detail-1.jpg`  | Linkes Bild im Duo (1000×750)                                    | `IMG_9936.jpg`                         |
| `praxis-detail-2.jpg`  | Rechtes Bild im Duo (1000×750)                                   | `IMG_9937.jpg`                         |
| `portrait.jpg`         | Portrait Veronika auf "Über mich" (900×1100, Hochformat)         | **Echtes Portrait von Veronika**       |

**Wichtig:** Die Dateinamen müssen **exakt** stimmen (Groß-/Kleinschreibung, .jpg statt .png).
